## 2026-07-13 详情源代码和匹配规则配色

服务器方面，Vicdata 是静态站，不需要数据库和后端计算。若短期继续使用 GitHub Pages，不买服务器也能访问，但国内访问速度受跨境链路影响，不能保证不同运营商和晚高峰稳定。若准备面向国内用户长期公开，内地服务器加 ICP 备案更稳；轻量服务器的主要作用是备案、托管静态文件和后续控制 Nginx 与日志。推荐配置是中国内地地域的 `2 核 2G` Linux 轻量服务器，华东地域更折中，带宽 `3Mbps` 到 `4Mbps` 可起步。阿里云看到的 `2 核 2G` 内地 Linux 一年优惠价 68 元可以考虑，域名在腾讯云、服务器在阿里云也可以备案，备案走服务器接入商阿里云，DNS 仍可用腾讯云解析。香港服务器不用备案，能比 GitHub Pages 更可控，但仍走跨境链路，国内流畅性取决于回国线路和带宽；如果目标是正式服务国内访问，内地服务器和备案仍是更合适的长期方案。

域名和服务器方案也在本轮做了判断。`vic3database.cn` 可以先注册，腾讯云页面显示首年 33 元、续费 38 元，成本低，适合先占住名称并完成域名实名认证。备案内容不宜弱化“游戏资料”属性，反而应明确写作《维多利亚 3》游戏资料查询站，说明只整理游戏内国家、地区、文化、公司、意识形态和地图资料，不提供游戏下载、充值、联机、论坛或用户发布服务，以免“意识形态”“地图”等词被误解为现实政治或现实地图内容。站内关于页和备案备注都应使用类似口径。
