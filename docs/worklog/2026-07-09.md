本轮提交记录如下：

若干视觉与交互问题已经修正。筛选栏滚动条改为常显，避免滚轮区域的状态不清楚。左上角白色块先后排查过滚动条和地图边缘，最终确认地图面板需要从顶栏和左侧边界内缩，已通过 `.map-panel` 的 `top: 12px` 与 `left: 12px` 修正，避免地图像素露到面板外。之后又确认 `#conceptTooltip` 的 `hidden` 属性被 `.concept-tooltip { display: grid; }` 覆盖时会造成空白提示框可见，因此新增 `.concept-tooltip[hidden] { display: none !important; }`，并把概念提示框改为鼠标停留 320 毫秒后再显示；鼠标移开会清除等待中的提示框，键盘聚焦仍可直接显示。

全站搜索已经改为独立弹窗。首页和各板块都保留右上角搜索入口，打开后显示遮罩、搜索输入框、旧版本搜索选项和结果列表。搜索结果已经使用统一实体徽标样式，便于在国家、地区、文化、公司、意识形态等对象之间快速区分。首页已经加入入口型内容，用于承接站点介绍、版本入口、更新记录和常用跳转。

筛选栏已经按新结构整理。国家板块的状态与类型合并为类型筛选，位阶、战略区域、传承、语言、传统等筛选继续保留；语言分区的小分区之间加入分割线。设置页已经接入“松散政权显示为白地”“过滤原住民语言、文化”“过滤松散政权 tag”等选项，并确认过滤松散政权 tag 使用 `country.countryType === "decentralized"` 判断，未按传承组判断，避免把伊基查、巴拉圭等非松散政权的原住民文化国家误处理。

页面布局已经改为左侧筛选栏、地图主体、右侧列表栏。原右侧详情栏改为独立详情页面，详情页会隐藏筛选栏和列表栏，并在标题区提供返回对应默认板块页面的按钮。国家列表等内容现在固定在右侧；折叠筛选栏时，列表栏不会再向左漂移遮挡地图工具按钮。地图工具栏保留折叠筛选、折叠列表和重置地图位置三个按钮，图标分别为 `text-search`、`layout-list`、`refresh-ccw`。旧的“筛选、列表、详情、深色/浅色”文字按钮已经从紧凑顶栏中移除；“开局世界局势”和地图图例文字行也已经移除。

导航结构已经从最初的左侧一级栏改为顶栏。顶栏左侧为 Vicdata 品牌入口，随后依次为国家、文化、地区、公司、意识形态；右侧为版本选择、全站搜索、设置和更新日志。旧的版本组选择已经移除，右上角只保留一个版本入口。Lucide 图标已经接入到站点外壳，当前使用 `database`、`earth`、`map`、`drama`、`briefcase-business`、`user-star`、`milestone`、`search`、`settings`、`refresh-ccw`、`text-search`、`layout-list` 等图标；前期路径问题已经改为直接使用 SVG 图片元素，避免 CSS 遮罩路径在浏览器中失效。

本轮确认的界面方向是：站点以 Vicdata 作为资料库入口，保留首页、国家、文化、地区、公司、意识形态等板块；首页承担站点入口、版本入口、全站搜索入口和更新记录入口。全站搜索与板块内搜索分离，右上角仅保留搜索图标，点击后打开遮罩搜索窗。词条识别采用统一视觉语言：公司、意识形态、地区特质使用图标；国家使用默认图标；地区使用圆形战略区域色块；文化、宗教使用方形或圆角矩形色块。整体配色暂按浅色米白、深色深棕的暖色方向处理，后续再继续细化。
