# Vicdata 蓝金浏览器图标设计

## 范围

本次制作 Vicdata 的应用、浏览器与页眉图标。大尺寸图形直接使用站名分层排版，小尺寸改用单个 `V`，不保留四字方阵。视觉气质来自深海军蓝、旧金色、浅金色细描边和单层金色外框；字形采用 Segoe Print，无衬线结构保持现代感。

## 图形规则

512、192 与 180 像素版本在深海军蓝圆角方形中排列三行 `Vic3`、`Data`、`Base`。三行文字均使用 Segoe Print 粗体，金色填充，并加极细浅金描边以形成金属边缘。文字按共同中轴居中排列，`Vic3` 最大，`Data` 与 `Base` 次之；180 像素版采用稍小字级和更紧凑行距，保留三行完整文字。每个大尺寸版本只有一层金色外框，圆角半径为画布宽度的 18%。

64、48、32 与 16 像素版本统一使用深海军蓝圆角方形底、单层金色外框和居中的金色 `V`。`V` 使用 Segoe Print 粗体，在安全区域内尽可能放大；不使用文字描边、缩写、方阵或其他装饰，以保证浏览器标签页和桌面图标中的边缘清楚。64 与 48 像素保留完整金色边框；32 与 16 像素的边框按画布比例减细，但继续可见。

深海军蓝为 `#102C3C`，主要金色为 `#D7AF4B`，大尺寸字形的浅金描边为 `#F2D17A`。PNG 背景保留透明圆角外侧。

## 输出与验收

输出可编辑 SVG 母版，以及 512、192、180、64、48、32 与 16 像素的 PNG 文件。网页使用 SVG 页眉图标以及 512、192、180、32 像素位图；16 像素用于多尺寸浏览器图标。验收时检查每个 PNG 的画布尺寸、透明圆角、深蓝底、金色边框、目标文字或 `V`，并在本地页面中确认 SVG 和清单图标引用正常。
