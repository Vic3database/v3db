# Vicdata 蓝金浏览器图标设计

## 范围

本次只制作 Vicdata 的应用与浏览器图标预览，不处理页眉完整标志、中文名称或英文名称。图形采用深海军蓝圆角方形底、金色字形和金色细边的组合，保持与现有站点图标的蓝金配色关联。

## 图形规则

512、192 与 180 像素版本在图标中央排列 `V 3 / D B` 四个字符。每个字符使用高对比衬线字形，字母与数字的笔画保留维多利亚时期印刷品的纵向感，但不加入花体、纹章或微小装饰。深海军蓝底色使用 `#102C3C`，主要金色使用 `#D7AF4B`，浅金色轮廓使用 `#F2D17A`。圆角半径按画布宽度的 18% 设置，金色双线边框仅用于 180 像素及以上版本。

64 与 48 像素版本继续使用四字方阵，但取消双线边框，并减少字母之间的空隙。32 与 16 像素版本只保留居中的金色大写 `V`，保持圆角深蓝底色，不保留边框和其他字符。小尺寸的 `V` 需有清晰的尖角与适度的字重，使其在浏览器标签页中仍能辨识。

## 输出与验收

预览阶段将输出可编辑的 SVG 母版，以及 512、192、180、64、48、32 与 16 像素的 PNG 文件。网页使用 512、192、180 与 32 像素图标；16 像素用于多尺寸浏览器图标。验收时检查每个 PNG 的画布尺寸、透明通道、圆角边缘和小尺寸 `V` 的辨识度，并在本地页面中确认图标引用没有失效。
