# 首页独立分类卡片设计

## 目标

首页的导航区域将外交、内政、经济、军事、社会、其他分别渲染为独立的分类卡片。卡片之间保留间距，分类标题和资料入口不再与相邻分类共用外框。

## 布局

导航区域继续使用六列网格。外交、内政、社会、其他各占三列；经济占四列；军事占两列。分类卡片内的入口数量与横向列数相同：三项分类为三列，经济为四列，军事为两列。入口内容、图标、待开放状态和既有的跳转行为保持不变。

分类卡片使用站点现有的低饱和金色边框与分隔线配色，沿用 `rgba(200, 164, 91, …)` 颜色体系；不使用高亮黄色。卡片底色、圆角和阴影沿用首页现有表面风格。

屏幕宽度低于当前首页的窄屏断点时，分类卡片收为单列；分类内部入口在中等宽度保留两列，在更窄屏幕收为一列。

## 验证

扩展 `scripts/check_homepage_layout.mjs`，静态检查六个独立分类卡片、六列占宽规则及标准配色变量。通过本地首页确认桌面端分类外框彼此独立，经济与军事同排的宽度为四列和两列，窄屏没有横向溢出。
