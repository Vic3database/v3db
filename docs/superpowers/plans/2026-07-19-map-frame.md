# Vicdata 地图外框实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让桌面地图四边保留一致的 12 像素外缘，并以细边框和 8 像素圆角裁切地图画布。

**Architecture:** 改动限定在桌面外壳样式。`map-panel` 决定地图在布局中的外缘，`map-viewport` 负责边框、圆角和对画布的裁切；画布继续铺满视口。新增一个静态检查，防止将来重新出现右、下贴边或圆角未裁切的问题。

**Tech Stack:** 原生 CSS、Node.js 静态检查脚本、现有浏览器验证流程。
