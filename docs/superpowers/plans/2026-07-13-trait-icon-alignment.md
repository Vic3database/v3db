# 特质配图与文字对齐实施计划

**目标：** 让地区特质、利益集团基础特质和国家风味利益集团特质显示对应游戏图标，并保持图标列与标题文字对齐。

**实现：** 在网站资源目录保存两类游戏图标；前端按特质键生成图标；完整卡片使用固定图标列和可换行文字列。静态检查验证资源、渲染链和样式合同。

## 步骤

1. 新增 `scripts/check_trait_icon_presentation.mjs`，断言图标资源映射、完整地区与利益集团特质卡片、风味渲染链及固定图标列存在。
2. 运行该检查，确认当前实现因缺少图标渲染和资源而失败。
3. 将 `game/gfx/interface/icons/state_trait_icons` 与 `game/gfx/interface/icons/ig_trait_icons` 的 PNG 复制至网站资源目录。
4. 修改 `site/app.js`，为地区特质效果和利益集团特质完整卡片插入带失败隐藏行为的图标；保留既有风味特质调用链。
5. 修改 `site/styles.css`，以固定 22 像素图标列、首行文字对齐和窄屏换行规则布局两类卡片。
6. 运行静态检查、语法检查和浏览器页面验证。
